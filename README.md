# DND5E-builder：传统PDF风格车卡器

这是我的第一个独立（其实是和AI一起）完成的完整项目，UI设计参考了最传统的pdf文档布局，可以实现导入FVTT、owlbear弗人插件、也可以保存为静态网页html或PDF。

本车卡器的设计初衷是为了方便 **面团玩家**/**纸质卡玩家**/**习惯pdf卡的玩家** 进行速记速查，没有做数据库支持自动填写表单，因此车卡需要搭配5e不全书复制粘贴使用。在车卡时法术、魔法物品等可以自由编辑注释，并在鼠标悬停的时候以浮窗形式出现，便于跑团中查看。

角色卡数据依赖于浏览器本地缓存，如果用户在设置中清除了“网站数据”或“浏览器存储”，存档可能会被一并删除。**建议在清理浏览器数据前先导出备份，以避免角色信息丢失。**

## 车卡注意事项

### 第一页

#### 攻击

点击 **+** 新建条目，可以从现有的法术、装备选择，或是直接新建武器。

武器可通过预设快速写入表单。

现有的法术/装备需要勾选 **保存为攻击武器/保存为伤害法术** 选项，才可在攻击面板中获取。

从攻击面板直接创建武器后，创建的武器会自动写入装备。

攻击项可通过双击打开对应的编辑弹窗，也可通过鼠标悬停（电脑）或单击（平板/手机）快速查看写入的描述。

攻击项可以右键删除。攻击项可以长按后拖拽重排序。

#### 装备

在灰色占位符处正常输入/粘贴文本，以，或、分割物品即可。按下回车后，整段文本将会被分割为多个物品。

每个物品均可双击打开编辑，除了添加描述外，同样可以给魔法物品添加特性。 [跳到魔法物品特性](#魔法物品特性)

物品可通过单击（平板/手机）或鼠标悬停（电脑）快速查看写入的描述。

物品可以右键或长按（平板/手机）打开菜单，选择删除或是移入库存（角色卡背面）。

#### 魔法物品特性

魔法物品可以填写多个特性，每个特性都会有一个次数栏，可以填写充能或是其他使用次数限制。

填写在次数栏的字段，可以从悬浮窗中直接快速编辑。

#### 角色特性与特质

输入方式与装备相同。

每个特性均可双击打开编辑，特性描述推荐从5e不全书直接复制。特性的浮窗会自动识别并高亮关键词。

对于存在多选项的特质（eg.战斗风格、超魔法选项），现在可以添加特质选项了。

对于想要自定义关键词的用户，可以从自定义项管理/特性关键词高亮进行编辑。

特性可以右键或长按（平板/手机）删除。


### 第二页

#### 角色信息

右上角的**1501DR**是可以编辑的，徽记上方的**信仰/组织/家族**同样也可以填写，搭配上传的徽记使用。当然，空着也是可以的。


### 第三页

#### 法术环位

法术环位默认根据第一页输入的职业与等级自动计算，但对于部分特殊子职（例如奥法骑士等），可以单击第三页上方的**施法职业栏**，编辑法术环位。

每环法术的区域可以通过长按拖拽调节显示长度（触摸屏可能在适配上存在一些问题）。

#### 法术

双击空白行即可填入/编辑法术。法术描述可以从5e不全书整段复制，浮窗会自动识别段落。

对于勾选**专注**的法术，专注标签会显示在行内右侧。单击可切换为专注中。

对于勾选**天生施法与其他来源施法**的法术，使用次数会显示在行内右侧，单击可快速修改。

法术项可以长按后拖拽重排序。


## 导入FVTT

1、在车卡器上完成车卡。

2、选择：导出文件 → 导出为fvtt json

3、在fvtt创建新角色，右键角色条目，选择导入数据。

4、选择第二步下载的json文件上传。

**法术、物品、职业等仍需从FVTT世界包拖入**。


## 导入枭熊（适配弗人插件）

1、在车卡器上完成车卡。

2、选择：导出文件 → 导出为枭熊 json

3、将导出的json文件直接拖到浏览器中，会在新界面直接显示文件内容，Ctrl+A全选后复制。

4、打开插件的角色卡界面（右侧菜单栏中），点击粘贴json，粘贴刚才复制的内容。

5、如果无法保存到房间，推荐利用上传空白excel卡来实现创建新角色，随后可以从新角色右上的导入json中直接选择先前保存的文件。
